    F: FnOnce(&mut dyn Connection) -> QueryResult<T>,
    Self: Sized, 

Opens a connection, which is valid inside the given handler closure..

This method does not open a transaction, and should used for operations not requiring transactions, e.g. single queries or schema mutations.
